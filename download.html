Ubuntu (and Ubuntu-based)

« PPA (stable releases) »

You can download the .deb file for your version and architecture from the link above and install it.

OR (preferred): Add the repository in your sources.list file with these easy steps:

$ sudo nano /etc/apt/sources.list

At the end of the file add:

deb http://ppa.launchpad.net/xsisqox/ppa/ubuntu YOUR_UBUNTU_VERSION_HERE main
deb-src http://ppa.launchpad.net/xsisqox/ppa/ubuntu YOUR_UBUNTU_VERSION_HERE main

Where YOUR_UBUNTU_VERSION_HERE is one of the following: hardy | intrepid | jaunty | karmic.

$ sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 8279047A
$ sudo aptitude update
$ sudo aptitude install viewnior

Crunchbang (9.04.1)

$ sudo aptitude install viewnior
