FAQ

  • What do i need to compile Viewnior?

    Viewnior depends only on gtk+-2.0 (it contains everything needed).
    Note that "Set as wallpaper" requires gconf2.

  • Is Viewnior available for Windows/MacOSX?

    No, at this time, no. Viewnior is made with portabilty in mind but is tested only under GNU/Linux. For OS X see (here)

  • How can I help?

    As on open-source software, Viewnior is nothing without You. So YES, you can help much - test, comment, tell everyone about us. Developers wishing to give a helping hand can contact me.

  • I have an idea/suggestion. How can I share it with you?

    You can file a bug, make a blueprint or ask a question at Launchpad

  • How can I translate Viewnior?

    You can do that at Launchpad.
    If you are interested in translating every new release, you can follow the feed for announces.

  • Where is "Set As Wallpaper"?

    It is an optional feature (disabled by default), which can be enabled by running ./configure --enable-wallpaper. Note that this works only under GNOME environment.
