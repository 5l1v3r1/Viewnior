Cool new things you didn't knew about Viewnior 0.4

Fullscreen & Slideshow

Press F11 and ... a whole new expirience! The toolbar is always there for you - shows current image's filename, nice easy controls - adjust timeout, turn on/off slideshow, zoom your image! You can exit either pressing "Leave Fullscreen", F11 or Escape key. In this mode, you can still zoom/drag/delete images.
To start/stop Slideshow you can check "Show next image ... " or just press Space from the keyboard. In Slideshow mode you cane still manually navigate using the buttons in the toolbar, or Left/Right keys.
Adjusting the seconds to switch the image is taking place immediately.

Navigate Easily

Too close? Can't make sense which part of the image you are seeing now? ... Just press the little "Move" icon at the lower right corner! Now drag the rectangle around ... If you use GIMP often, you won't face any problems.

Comfortable zoom?

Want to zoom 1:1 in just a mouse click? Double click the newly opened image in the area you want to zoom! Want to fit the image back in the window? Double click again! ... Addicting, isn't it?

Close to you

Want to delete an image you are viewing, but it is in a folder with many, many other images, so browsing doesn't seem the option? Just press Delete key and confirm!
Viewing a potentially good wallpaper? File -> Set as wallpaper will do the work only if you use Gnome.
Note, that for this option to be available, you need to run the configure script with "./configure --enable wallpaper"

Adjust window size

Want your picture to perfectly fit the window? Maybe resizing the window is the key! Clicking "Adjust window size" will make the window look like you just opened the image - no redundant space! Isn't that what we all want? More Screen space!
